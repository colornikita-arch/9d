Частные случаи параллелограмма

Прямоугольник

Прямоугольник - параллелограмм, у которого все углы прямые.

$\angle A = \angle B = \angle C = \angle D = 90^{\circ}$

Прямоугольник - частный случай параллелограмма, поэтому обладает всеми свойствами параллелограмма

Свойства прямоугольника

Дополнительное свойство прямоугольника:

Диагонали прямоугольника равны

Дано: $ABCD$ - прямоугольник
$AC \cap BD = O$

Доказательство:

1. Рассмотрим прямоугольник $ABCD$.

2. $\triangle ABD$ и $\triangle DCA$:

$AB = DC$ (противоположные стороны)

$AD$ - общая сторона

$\angle BAD = \angle CDA = 90^{\circ}$.

3. $\triangle ABD = \triangle DCA$ по двум сторонам и углу между ними.

4. Следовательно, $BD = AC$.

Признаки прямоугольника

1. Если в параллелограмме один из углов прямой, то этот параллелограмм является прямоугольником.

Дано: $ABCD$ - параллелограмм
$\angle BAD = 90^{\circ}$

Доказательство:

1. По свойству параллелограмма противоположные углы равны, значит $\angle BCD = \angle BAD = 90^{\circ}$.

2. По свойству $AB \Vert CD$ и секущей $AD$: $\angle BAD + \angle CDA = 180^{\circ}$ (односторонние углы).

3. $\angle CDA = 180^{\circ} - \angle BAD = 180^{\circ} - 90^{\circ} = 90^{\circ}$.

4. Сумма углов четырёхугольника равна $360^{\circ} \Rightarrow$
$\Rightarrow \angle ABD = 360^{\circ} - \angle BAD - \angle ADC - \angle BCD =$
$ = 360^{\circ} - 90^{\circ} - 90^{\circ} - 90^{\circ} = 90^{\circ}$.

Признаки прямоугольника

2. Если в четырёхугольнике три угла прямые, то этот четырёхугольник является прямоугольником.

Дано: $ABCD$ - четырёхугольник
$\angle A = \angle B = \angle C = 90^{\circ}$

Доказательство:

1. Сумма углов четырёхугольника равна $360^{\circ} \Rightarrow$
$\Rightarrow \angle ADC = 360^{\circ} - \angle BAD - \angle ABC - \angle BCD =$
$ = 360^{\circ} - 90^{\circ} - 90^{\circ} - 90^{\circ} = 90^{\circ}$.

2. По признаку $ABCD$ - параллелограмм (противоположные углы попарно равны).

3. $ABCD$ - параллелограмм, $\angle BAD = 90^{\circ} \Rightarrow$
$\Rightarrow ABCD$ - прямоугольник по признаку (параллелограмм с прямым углом).

Признаки прямоугольника

3. Если в параллелограмме диагонали равны, то этот параллелограмм является прямоугольником.

Дано: $ABCD$ - параллелограмм
$AC = BD$

Доказательство:

1. По свойству параллелограмма диагонали точкой пересечения делятся пополам: $AO = OC, ~ BO = OD$.

2. $AC = BD, ~ AO = OC, ~ BO = OD \Rightarrow$ $\Rightarrow AO = OC = BO = OD$.

3. $\triangle ABC: BO = AO = OC = \frac{1}{2}AC \Rightarrow$ $\Rightarrow \angle ABC = 90^{\circ}$ - по признаку прямоугольного треугольника.

4. $ABCD$ - параллелограмм, $\angle ABC = 90^{\circ} \Rightarrow ABCD$ - прямоугольник по признаку (параллелограмм с прямым углом).

Ромб

Ромб - параллелограмм, у которого все стороны равны.

$AB = BC = CD = DA$

Ромб - частный случай параллелограмма, поэтому обладает всеми свойствами параллелограмма

Свойства ромба

Дополнительнык свойства ромба:

1. Диагонали являются биссектрисами углов

2. Диагонали перпендикулярны друг другу

Дано: $ABCD$ - ромб

Доказательство:

1. Диагонали - биссектрисы:

$\triangle ABD$ - равнобедренный ($AB = AD$ как стороны ромба)

$AO$ - медиана $\Rightarrow$ $AO$ - биссектриса

2. Диагонали перпендикулярны:

$\triangle AOB$ и $\triangle AOD$:

$AB = AD$ (стороны ромба)

$AO$ - общая

$OB = OD$ (диагонали параллелограмма точкой пересечения делятся пополам)

$\triangle AOB = \triangle AOD \Rightarrow \angle AOB = \angle AOD = 90^{\circ}$

Признаки ромба

1. Если в параллелограмме диагонали перпендикулярны друг другу, то этот параллелограмм является ромбом.

Дано: $ABCD$ - параллелограмм
$AD \bot BC$

Доказательство:

1. По свойству параллелограмма (диагонали точкой пересечения делятся пополам): $AO = OC, ~ BO = OD$.

2. По признаку равенства прямоугольных треугольников (по двум катетам): $\triangle AOB = \triangle COD = \triangle COD = \triangle AOD$.

3. Т.к. в равных треугольниках соответственные элементы равны, то $AB = BC = CD = DA$.

4. $ABCD$ - параллелограмм и $AB = BC = CD = DA$, тогда $ABCD$ - ромб по определению.

Признаки ромба

2. Если в параллелограмме диагонали являются биссектрисами углов, то этот параллелограмм является ромбом.

Дано: $ABCD$ - параллелограмм
$BD$ - биссектриса $\angle ABC$
$BD$ - биссектриса $\angle ADC$

Доказательство:

1. По свойству параллелограмма (диагонали точкой пересечения делятся пополам): $AO = OC, ~ BO = OD$.

2. По свойству параллелограмма (противоположные углы равны): $\angle ABC = \angle ADC$
$BD$ - биссектриса углов $\angle ABC$ и $\angle ADC$,
значит $\angle ABO = \angle CBO = \angle CDO = \angle ADO$.

3. По признаку равенства треугольников (по двум сторонам и углу между ними): $\triangle AOB = \triangle COD = \triangle COD = \triangle AOD$.

3. Т.к. в равных треугольниках соответственные элементы равны, то $AB = BC = CD = DA$.

4. $ABCD$ - параллелограмм, $AB = BC = CD = DA$, тогда $ABCD$ - ромб по определению.

Квадрат

Квадрат - правильный четырёхугольник, у которого все стороны и углы равны.

$AB = BC = CD = DA$

$\angle A = \angle B = \angle C = \angle D$

Квадрат - частный случай прямоугольника и ромба, а значит обладает всеми свойствами прямоугольника и ромба

Свойства квадрата

Свойства:

1. Все стороны квадрата равны.

2. Все углы квадрата прямые.

3. Диагонали квадрата равны.

4. Диагонали квадрата являются биссектрисами углов квадрата.

5. Диагонали точкой пересечения делятся пополам.

6. Диагонали перпендикулярны друг другу.

Признаки квадрата

1. Если в прямоугольнике соседние стороны равны, то этот прямоугольник является квадаратом.

Дано: $ABCD$ - прямоугольник
$AB = BC$

Доказательство:

1. $ABCD$ - прямоугольник (соответственно и параллелограмм), тогда по свойству параллелограмма (противоположные стороны равны) $AB = CD, ~ BC = AD$.

2. Т.к. $AB = BC$, то $AB = BC = CD = AD$.

3. $AB = BC$, то $AB = BC = CD = AD$.

4. $ABCD$ - прямоугольник, тогда по определению прямоугольника $\angle A = \angle B = \angle C = \angle D = 90^{\circ}$.

5. $(3) + (4) \Rightarrow ABCD$ - квадрат по определению.

Признаки квадрата

2. Если диагонали прямоугольника перпендикулярны, то этот прямоугольник является квадаратом.

Дано: $ABCD$ - прямоугольник
$AC \bot BD$

Доказательство:

1. По свойству параллелограмма (диагонали точкой пересечения делятся пополам): $AO = OC, ~ BO = OD$.

2. По свойству прямоугольника (диагонали равны) $AC = BD$.

3. $(1) + (2) \Rightarrow AO = BO = CO = DO$.

4. По признаку равенства прямоугольных треугольников (по двум катетам): $\triangle AOB = \triangle COD = \triangle COD = \triangle AOD$.

5. Т.к. в равных треугольниках соответственные элементы равны, то $AB = BC = CD = DA$.

6. $ABCD$ - прямоугольник, тогда по определению прямоугольника $\angle A = \angle B = \angle C = \angle D = 90^{\circ}$.

5. $(5) + (6) \Rightarrow ABCD$ - квадрат по определению.

Признаки квадрата

3. Если один из углов ромба прямой, то этот ромб является квадаратом.

Дано: $ABCD$ - ромб
$\angle A = 90^{\circ}$

Доказательство:

1. По свойству параллелограмма (противоположные углы равны): $\angle BCD = \angle BAD = 90^{\circ}$.

2. Сумма углов четырёхугольника равна $360^{\circ}$, значит $\angle ABC + \angle ADC = 360^{\circ} - \angle BCD - \angle BAD =$
$= 360^{\circ} - 90^{\circ} - 90^{\circ} = 180^{\circ}$

3. По свойству параллелограмма (противоположные углы равны): $\angle ABC = \angle ADC$.

4. $(2)+(3) \Rightarrow \angle ABC = \angle ADC = \frac{180^{\circ}}{2} = 90^{\circ}$.

5. По определению ромба $AB = BC = CD = DA$; $\angle BAD = \angle ABC = \angle BCD = \angle CDA = 90^{\circ}$
$ABCD$ - квадрат по определению.

Признаки квадрата

4. Если диагонали ромба равны, то этот ромб является квадаратом.

Дано: $ABCD$ - ромб
$AC = BD$

Доказательство:

1. По свойству параллелограмма (диагонали точкой пересечения делятся пополам): $AO = OC$.

2. По свойству ромба (диагонали перпендикулярны друг другу) $\angle AOB = \angle BOC = 90^{\circ}$

3. По признаку равенства прямоугольных треугольников (по двум катетам): $\triangle AOB = \triangle BOC$.

4. $\triangle AOB: AO = OB \Rightarrow \triangle AOB$ - равнобедренный.

5. По свойству равнобедренного $\triangle AOB:$ $\angle OAB = \angle OBA$.

6. Сумма углов треугольника равна $180^{\circ}:$ $\angle OAB + \angle OBA = 180^{\circ} - \angle AOB = 180^{\circ} - 90^{\circ} = 90^{\circ}$.

7. $(5) + (6) \Rightarrow \angle OAB = \angle OBA = 45^{\circ}$.

8. Т.к. в равных треугольниках соответственные элементы равны, а $\triangle AOB = \triangle BOC$, то $\angle OBA = \angle OBC = 45^{\circ} \Rightarrow \angle ABC = 90^{\circ}$.

9. По признаку квадрата (один угол ромба прямой) $ABCD$ - квадрат.

Трапеция

Трапеция - четырёхугольник, у которого две стороны параллельны, а две другие нет.

$BC \Vert AD$

$AB \nparallel CD$

Равнобокая трапеция

Равнобокая трапеция - трапеция, у которой боковые стороны равны.

$BC \Vert AD$

$AB \nparallel CD$

$AB = CD$

Свойства равнобокой трапеции

1. Углы при основаниях равнобокой трапеции равны.

Дано: $ABCD$ - равнобокая трапеция

Доказательство:

1. $BH \bot AD, ~ CM \bot AD ~ \Rightarrow BH ~ \Vert CM$
по признаку $BH \Vert CM$ и секущей $AD$ (односторонние углы $\angle BHM$, $\angle CMD$ равны).

2. $BH \Vert CM, ~ BC \Vert AD, ~ \angle BHM = 90^{\circ} ~ \Rightarrow BHMC$ - прямоугольник по признаку.

3. $BHMC$ - прямоугольник $\Rightarrow BH = CM$ - по свойству прямоугольника.

4. По признаку равенства прямоугольных треугольников (по катету и гипотенузе) $\triangle ABH = \triangle DCM$ $(AB = CD, ~ BH = CM)$.

5. В равных треугольниках соответственные элементы равны, значит $\angle BAH = \angle CDM$.

Свойства равнобокой трапеции

2. Диагонали равнобокой трапеции равны.

Дано: $ABCD$ - равнобокая трапеция

Доказательство:

1. $AB = CD, ~ AD$ - общая,
$\angle BAD = \angle CDA$ - по свойству р/б трапеции, значит $\triangle ABD = \triangle DCA$ по признаку равенства треугольников (по двум сторонам и углу между ними).

2. В равных треугольниках соответственные элементы равны, значит $BD = AC$.

Признаки равнобокой трапеции

1. Если углы при некотором основании трапеции равны, то она равнобокая.

Дано: $ABCD$ - трапеция
$\angle BAD = \angle CDA$

Доказательство:

1. $BH \bot AD, ~ CM \bot AD ~ \Rightarrow BH ~ \Vert CM$
по признаку $BH \Vert CM$ и секущей $AD$ (односторонние углы $\angle BHM$, $\angle CMD$ равны).

2. $BH \Vert CM, ~ BC \Vert AD, ~ \angle BHM = 90^{\circ} ~ \Rightarrow BHMC$ - прямоугольник по признаку.

3. $BHMC$ - прямоугольник $\Rightarrow BH = CM$ - по свойству прямоугольника.

4. По признаку равенства прямоугольных треугольников (по катету и острому углу) $\triangle ABH = \triangle DCM$ $(\angle BAH = \angle CDM, ~ BH = CD)$.

5. В равных треугольниках соответственные элементы равны, значит $AB = CD$.

Признаки равнобокой трапеции

2. Если диагонали трапеции равны, то она равнобокая.

Дано: $ABCD$ - трапеция
$AC = BD$

Доказательство:

1. $BH \bot AD, ~ CM \bot AD ~ \Rightarrow BH ~ \Vert CM$
по признаку $BH \Vert CM$ и секущей $AD$ (односторонние углы $\angle BHM$, $\angle CMD$ равны).

2. $BH \Vert CM, ~ BC \Vert AD, ~ \angle BHM = 90^{\circ} ~ \Rightarrow BHMC$ - прямоугольник по признаку.

3. $BHMC$ - прямоугольник $\Rightarrow BH = CM$ - по свойству прямоугольника.

4. По признаку равенства прямоугольных треугольников (по катету и гипотенузе) $\triangle BHD = \triangle CMA$ $(BD = AC, ~ BH = CM)$.

5. В равных треугольниках соответственные элементы равны, значит $\angle BDH = \angle CAM$.

6. По признаку равенства треугольников (по двум сторонам и углу между ними): $\triangle BAD = \triangle CDA$ $(BD = CA, ~ AD$ общая, $\angle BDA = \angle CAD)$.

7. Т.к в равных треугольниках соответственные элементы равны: $AB = CD$.
